# AiluxOS 旧版首页首屏参考

首屏为深色沉浸式背景，视觉主题偏生物科技与神经网络纹理，整体采用深蓝黑底色、中心聚焦式主标题与发光按钮。

## 首屏信息架构

| 区域 | 观察结果 |
|---|---|
| 顶部左侧 | 品牌标识为 AiluxOS，带小型图标与文字组合 |
| 顶部中部 | 有 `Platform` 与 `Applications` 导航项 |
| 顶部右侧 | 有中英文切换与登录入口 |
| 首屏徽标 | 小胶囊文案为 `AI-NATIVE BIOPHARMA R&D OS` |
| 主标题 | `AiluxOS` 与 `Lighting the Path to Cures` 分层展示，强调渐变高光 |
| 副标题 | 中文说明聚焦 AiluxAgent、LuxSight、AtlaX 数据平台整合能力 |
| 主按钮 | 主按钮为“探索 AiluxAgent”，这是最关键入口 |
| 次按钮 | 次按钮为“登录平台” |

## 对当前项目的可复用结论

1. 可以为当前项目新增一个独立首页，保留深色生物科技风格，但不追求逐像素复刻。
2. 首页的核心 CTA 应是进入 Agent 工作台，因此“探索 AiluxAgent”应直接跳转到当前 Agent 页面。
3. 当前已有中英文切换机制，可把首页也接入同一语言状态。
4. 最稳妥的路由方式是让 `/` 作为首页，让现有 Agent 工作台迁移到 `/agent`。

## 当前实现预览验证

已在开发预览中确认根路由展示新的深色首页首屏，包含品牌导航、主标题、主 CTA“探索 AiluxAgent”以及架构与应用区块。

已验证点击首页主 CTA 后可成功跳转至 `/agent`，并进入当前多栏 Agent 工作台界面。
